This is the Henry theme for the [Zola](https://www.getzola.org/) static site generator. Henry is based on the original styles of Jekyll, and the name is the first name of [Dr. Henry Jekyll](https://en.wikipedia.org/wiki/Dr._Jekyll_and_Mr._Hyde_(character)).

The theme is licensed under the [MIT license](https://opensource.org/licenses/MIT), so you may freely use and redistribute either for commercial or non-commercial work.

You can find the source code for the theme in the [sirodoht/zola-henry](https://www.github.com/sirodoht/henry) repository.
For a description of this theme and links to other Zola themes, please go to <https://www.getzola.org/themes/>.
